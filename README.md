# UniAcco Assignment

## Clone Repositoty

```
git clone https://github.com/bhaunkushwah/uniacco-task.git
```

## Backend

```
1. cd uniacco-task
2. cd backend
3. virtualvenv env
4. Linux: source env/bin/activate
5. pip3 install -r requirements.txt
6. python3 manage.py runserver
```

## Frontend

```
1. cd uniacco-task
2. cd frontend
3. npm install
4. npm start
```
###

Go to http://localhost:3000
